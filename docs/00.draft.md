# 혈당 스파이크 경고 플랫폼 기획서

## 1. 프로젝트 개요

### 1-1. 기획 배경

말토덱스트린, 액상과당, 정제 전분 등은 '당이 아닌 것처럼 보이지만' 실제로는 혈당을 매우 빠르게 상승시키는 물질이다.
일반 소비자는 성분표를 봐도 위험도를 직관적으로 이해하기 어렵고, 기존 정보는 과장되거나 공포 마케팅에 치우친 경우가 많다.

본 프로젝트는 **감정 배제 · 과장 최소 · 생리학/영양학 기반**으로
혈당을 빠르게 올리는 물질의 '정체와 작동 방식'을 설명하고,
일상에서 스스로 판단할 수 있는 기준을 제공하는 것을 목표로 한다.

---

### 1-2. 미션 & 비전

* **미션**: 혈당 스파이크를 '느낌'이 아닌 '구조'로 이해하게 만든다.
* **비전**: 가공식품과 생리 반응을 구조적으로 설명하는 장기 지식 플랫폼

---

## 2. 타겟 정의

### 2-1. 핵심 타겟

* 공복혈당 / 식후혈당이 경계선에 있는 사람
* 당뇨 전단계, 인슐린 저항성에 관심 있는 일반인
* 다이어트 중인데 식단 관리가 반복적으로 실패하는 사람
* "설탕은 안 먹는데 혈당이 오른다"고 느끼는 사람

### 2-2. 확장 타겟

* 부모(아이 간식 성분 분석)
* 운동·헬스 유저(보충제·가공 단백질 성분)
* 반려동물 보호자(사료·간식 가공 방식)

---

## 3. 전체 구조 (유입 → 신뢰 → 참여)

```
블로그(정중·정확) ┐
                      → 신뢰 형성 → 앱/웹 유입 → 정보 교류
유튜브 쇼츠(재미) ┘
```

---

## 4. 콘텐츠 전략

### 4-1. 블로그 (신뢰 중심)

**톤 & 원칙**

* 감정 표현 최소화
* '나쁘다' 대신 '흡수 속도·구조'로 설명
* 결론보다 메커니즘 우선

**주요 카테고리**

1. 탄수화물·당류 구조 분류
2. 가공식품 성분 사전
3. 가공 방식이 만드는 차이
4. 실험·관찰 기반 기록

모든 글 하단에 **구조 요약 + 앱/웹 안내** 삽입

---

### 4-2. 유튜브 쇼츠 (AI 강아지 인터뷰)

**포맷 목적**

* 설명 ❌
* 호기심 생성 ⭕
* 질문 유도 ⭕

**기본 흐름**

* 강아지 질문 → 짧은 팩트 → 구조 힌트 → 앱 유도

---

## 5. 앱 / 웹 기획

### 5-1. 핵심 기능

1. 가공식품 성분·가공 방식 사전
2. 구조 비교 뷰 (성분·가공·반응)
3. 사용자 관찰 기록 (선택)
4. 질문 중심 커뮤니티

### 5-2. 운영 원칙

* 의료 진단 ❌
* 제품 판매 ❌
* 공포 마케팅 ❌

판단 기준 제공에만 집중

---

## 6. 신뢰 전략

* 교과서·생리학 기반 출처 명시
* 단정적 표현 제한
* 커뮤니티 가이드라인 엄격 적용

---

## 7. 확장 로드맵

**Phase 1**: 기획·콘텐츠 축적
**Phase 2**: 구조화·비교 기능
**Phase 3**: 식품 전반·반려동물 영역 확장

---

## 8. 요약

이 프로젝트는 특정 성분을 비난하는 플랫폼이 아니라,
**가공이 생리 반응을 어떻게 바꾸는지를 구조로 설명하는 장기 지식 프로젝트**다.
